# CRITÉRIOS DE ACEITAÇÃO
## Hospital Agostinho Neto - Plataforma de Marcação de Consultas

---

## US01 - Marcação de Consulta (5 critérios)

| # | Critério de Aceitação |
|---|----------------------|
| CA01 | O paciente deve conseguir escolher a especialidade médica |
| CA02 | O paciente deve conseguir ver os médicos disponíveis |
| CA03 | O paciente deve conseguir ver os horários livres |
| CA04 | O sistema deve enviar confirmação por mensagem |
| CA05 | O sistema deve atualizar a agenda do médico automaticamente |

---

## US02 - Reagendamento de Consulta (5 critérios)

| # | Critério de Aceitação |
|---|----------------------|
| CA06 | O paciente deve conseguir visualizar sua consulta atual |
| CA07 | O paciente deve conseguir escolher uma nova data/hora |
| CA08 | O sistema deve avisar se há conflitos de horário |
| CA09 | O sistema deve enviar nova confirmação |
| CA10 | O sistema deve liberar o horário antigo |

---

## US03 - Cancelamento de Consulta (5 critérios)

| # | Critério de Aceitação |
|---|----------------------|
| CA11 | O paciente deve conseguir cancelar com mais de 24h de antecedência |
| CA12 | O sistema deve confirmar o cancelamento |
| CA13 | O sistema deve liberar o horário automaticamente |
| CA14 | O sistema deve impedir cancelamento com menos de 24h |
| CA15 | O sistema deve notificar o paciente do cancelamento |

---

## US04 - Triagem Automática (5 critérios)

| # | Critério de Aceitação |
|---|----------------------|
| CA16 | O sistema deve perguntar sobre os sintomas |
| CA17 | O sistema deve classificar a prioridade (Urgente/Alta/Média/Baixa) |
| CA18 | Pacientes urgentes devem ser atendidos primeiro |
| CA19 | O médico deve receber a classificação |
| CA20 | O sistema deve sugerir encaminhamento adequado |

---

## US05 - Gestão de Médicos (5 critérios)

| # | Critério de Aceitação |
|---|----------------------|
| CA21 | O administrador deve conseguir cadastrar novos médicos |
| CA22 | O administrador deve conseguir definir horários de trabalho |
| CA23 | O administrador deve conseguir atualizar especialidades |
| CA24 | O administrador deve conseguir bloquear dias de folga |
| CA25 | O administrador deve conseguir visualizar a agenda do médico |
